# お問い合わせフォーム（サンプル）

## 概要
HTML / CSS / JavaScript を使用して作成した、入力チェック付きのお問い合わせフォームです。  
Web制作の基礎（フォーム・バリデーション・DOM操作・レスポンシブ対応）の理解を目的として制作しました。

## デモURL
https://vazisan.github.io/contact-form-sample/

## 使用技術
- HTML
- CSS
- JavaScript（Vanilla JS）
- Git / GitHub

## 機能
- 名前・件名・メッセージの必須入力チェック
- 件名で「その他」を選択した場合のみ入力欄を表示
- メッセージ100文字制限
- 残り文字数のリアルタイム表示
- エラーメッセージの表示
- スマートフォン対応（レスポンシブ）

## 工夫した点
- JavaScriptのみでフォームバリデーションを実装
- ユーザーが迷わないようエラーメッセージを項目ごとに表示
- スマホでも操作しやすいレイアウトを意識
