# お問い合わせフォーム（サンプル）

## 概要

HTML / CSS / JavaScript を使用して作成した、**入力チェック付きのお問い合わせフォーム**です。
Web制作の基礎理解と JavaScript の学習成果として作成しました。

## 制作目的

* HTML / CSS / JavaScript の役割を理解する
* JavaScript によるフォームバリデーションを実装する
* 実務を意識したシンプルで使いやすい UI を作る

## 使用技術

* HTML5
* CSS3
* JavaScript（Vanilla JS）

## 機能一覧

* 名前・件名・メッセージの必須入力チェック
* 件名「その他」選択時の入力欄表示切り替え
* メッセージ100文字制限
* 残り文字数のリアルタイム表示
* エラーメッセージの表示
* レスポンシブ対応（スマートフォン表示）

## 工夫した点

* JavaScript で DOM 操作を行い、ユーザーの操作に応じて表示を切り替えるようにしました
* エラー内容が分かりやすくなるよう、項目ごとにメッセージを表示しています
* 初心者でも読みやすいよう、コードはできるだけシンプルに記述しました

## 学んだこと

* form 要素の基本構造
* addEventListener を使ったイベント処理
* 入力値の取得と条件分岐
* Git / GitHub を使ったコード管理と公開
