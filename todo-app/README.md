# Todoアプリ

## 概要
JavaScriptのDOM操作の理解を目的に作成したシンプルなTodoアプリです。

## デモURL
https://vazisan.github.io/contact-form-sample/todo-app/

## 機能
- Todoの追加
- Enterキーで追加
- クリックで完了状態の切り替え
- Todoの削除

## 使用技術
- HTML
- CSS
- JavaScript（Vanilla JS）

## 学習ポイント
- DOM操作（createElement / appendChild）
- イベント処理（click / keydown）
- classList を使った状態管理
